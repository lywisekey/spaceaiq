# spaceaiq_experience_system.md

## Core Design Principle

Design the website around the logo.

Do not force the logo to fit SpaceX.

The website must evolve from the geometry, energy, and symbolism already present in the SPACEAIQ mark.

---

# Logo DNA Analysis

The SPACEAIQ logo communicates:

- Velocity
- Aerospace engineering
- Orbital movement
- Precision manufacturing
- Quantum pathways
- Strategic infrastructure

Important geometric cues:

A = launch vector

E = orbital layers

Q = gateway / network node

These shapes become recurring visual motifs throughout the site.

Never introduce unrelated visual languages.

---

# Brand Personality

SPACEAIQ is:

- Strategic
- Ambitious
- Precise
- Sovereign
- Technological

Not:

- Friendly startup
- Consumer app
- Crypto project
- AI wrapper
- SaaS dashboard

The emotional response should be:

"We are witnessing the construction of a planetary-scale system."

---

# Visual Style

Reference:

40% SpaceX
25% Apple
20% Anduril
15% Palantir

Not visually identical.

Only borrow principles.

---

# Color Architecture

Primary Canvas:

#000000

Primary Text:

#FFFFFF

Brand Accent:

SPACEAIQ Red

Color Distribution:

Black: 70%

White: 20%

Red: 10%

The logo becomes the strongest red object on every screen.

Nothing should compete with the logo.

---

# Signature Motifs

Inspired directly from logo geometry.

Use:

- diagonal launch vectors
- orbital trajectories
- converging network lines
- angular aerospace forms

Avoid:

- random circles
- blob shapes
- glassmorphism
- neumorphism

All geometry must feel engineered.

---

# Hero Design

Goal:

Create awe.

Structure:

[small label]

THE QUANTUM HIGHWAY

One sentence.

One CTA.

One visual.

No feature list.

No trust badges.

No statistics above the fold.

---

# Hero Background System

Preferred order:

1. Earth at night from orbit
2. Satellite constellation
3. Quantum network visualization
4. Deep space imagery

Backgrounds must feel real.

Never use generic AI art.

---

# Navigation

Extremely minimal.

Logo left.

4–5 links maximum.

No boxed menu items.

No bright hover animations.

Hover behavior:

opacity transition only.

---

# Section Storytelling Model

Every section must answer one question.

Example:

What is the Quantum Highway?

How does it work?

Why does it matter?

Who benefits?

Why now?

One section.
One idea.

---

# Typography

Headlines should feel monumental.

Examples:

CONNECTING THE PLANET.

BUILDING THE QUANTUM HIGHWAY.

SECURING GLOBAL COMMUNICATIONS.

Short.

Strong.

Memorable.

---

# Content Density

Reduce content by 50%.

If a paragraph exceeds four lines:

rewrite it.

If a section contains more than one CTA:

remove one.

---

# Motion System

Animation should feel like orbital mechanics.

Motion language:

- glide
- reveal
- tracking
- trajectory

Never:

- bounce
- shake
- spin

---

# Premium Experience Rules

Every page should pass this test:

Would this feel credible if presented to:

- NASA
- ESA
- DARPA
- National governments
- Institutional investors

If not:

simplify.

---

# Final Design Test

The website should feel like:

"The operating system of future space communications."

Not:

"An AI company website."

That distinction is critical.
